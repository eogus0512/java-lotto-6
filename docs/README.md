## 기능 목록

- 로또 도메인 로직 생성
    - 로또 번호 변수
- 로또 번호 비교 기능
    - 로또 객체끼리 로또 번호 비교하고 같은 로또 개수를 반환한다.
- 로또 구입 금액 입력 기능
    - 사용자로부터 금액 입력을 입력 받는다.
    - 금액을 1000으로 나누어 로또 수량을 얻는다.
- 당첨 번호 입력 기능
    - 사용자로부터 당첨 번호를 입력받는다.
    - 입력 받은 문자열을 ',' 기준으로 split한다.
    - 사용자로부터 보너스 번호를 입력받는다.
- 구입한 로또 수량 만큼 로또 발행
    - 수량만큼 크기 6의 랜덤 숫자 배열 생성
- 당첨 통계 출력 기능
